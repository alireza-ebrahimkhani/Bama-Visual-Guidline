Social Media

Bama / BrandExpression / Social Media

Media Icon

For social media profile picture, place a black sign on an green background. The container’s corner radius can flexibly change based on the media.

برای تصویر نمایه شبکه‌های اجتماعی، نشانه‌ با رنگ مشکی را روی یک صفحه‌ی سبز قرار دهید. بسته به شرایط گوشه‌های صفحه می‌تواند به اندازه‌های مختلف گرد شود.

Sample Post Types

Use a combination of illustration, motion graphics and photos for social media posts.

برای پست‌های شبکه‌های اجتماعی از ترکیب تصویرسازی، موشن‌گرافیک و عکس استفاده کنید.

Sample story Types

Use a variety of illustrations, motion graphics and engaging images for social media stories.

برای استوری‌های شبکه‌های اجتماعی نیز به طور متنوع از تصویرسازی، موشن‌گرافیک و تصاویر جذاب استفاده کنید.

Highlights

You can use illustrations, photos and icons to design highlight covers. When using images, make sure that the visible part of image (croped area) is slightly detailed so that it can be seen clearly.

برای طراحی هایلایت‌ها می‌توانید از تصویرسازی، عکس و آیکن‌ها استفاده کنید. در شرایطی که از عکس برای هایلایت استفاده می‌کنید دقت داشته باشید تا تصویر نهایی جزئیات کمی داشته باشد تا به وضوح دیده شود.
